## Recommended Fixes

1. **Homepage h1 test failure:** The homepage does not have an `h1`. It uses `h2` for article titles. Update the test selector to look for the page title element, or add an `h1` to the homepage if one is desired.

2. **Mobile nav strict mode issue:** Fix the locator to target a specific nav: `page.locator('nav.top-bar')` instead of `page.locator('nav, .nav')`.

3. **Non-www domain duplicates:** Pages contain links to `successionholdingllc.com` (non-www) variant URLs. These resolve but create duplicate content issues. Consider updating internal links to use relative paths or the www variant for canonical consistency.

4. **Cloudflare email-protection 404s:** These are expected — Cloudflare automatically injects email encoding on mailto links. No fix needed.
